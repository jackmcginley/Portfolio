Structured Tree

General purpose JS component

Description: This creates a structured tree with all of the node and parent-child branch relationships. This type of component can be widely used for a data structure with multiple branches. The component also provides dynamic rendering of the tree as nodes are added and removed.

Technologies used: HTML5, CSS3, Javascript

Notable features: Object-oriented Javascript - User interface to build and render a tree structure on-the-fly

Biggest challenges: Firstly the tree algorithm had to be designed, which proved to be quite complex. I was also keen to implement this using an OO approach so learning how to do that was also something new. Lastly, the biggest challenge is probably the algorithm to render the tree in the correct hierarchy. The layout required an adaptive algorithm that takes account of both the depth of the tree and its width.

The site can be viewed here and the code on Github is here.
